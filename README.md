# Serverless Image Resizer

A cloud-based image upload and resize app using AWS.

## Features
- Upload image from frontend
- Store original image in S3
- Auto resize using Lambda
- Save resized image in S3


## Tech Stack
- HTML / CSS / JavaScript (Frontend)
- Python (AWS Lambda)
- AWS S3
- AWS Lambda
- API Gateway
